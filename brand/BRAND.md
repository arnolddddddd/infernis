# INFERNIS Brand Guidelines

## Logo Files

| File | Use |
|------|-----|
| `infernis-logo.svg` | Primary logo — light backgrounds |
| `infernis-logo-dark.svg` | Dark background variant |
| `infernis-icon.svg` | Standalone flame icon (favicons, avatars, small sizes) |

## Colors

### Primary Palette

| Name | Hex | Usage |
|------|-----|-------|
| Flame Orange | `#F97316` | Primary brand color, CTAs |
| Flame Red | `#EF4444` | Accent, alerts |
| Flame Yellow | `#FCD34D` | Highlights, inner flame |
| Slate Dark | `#1E293B` | Text on light backgrounds |
| Slate Light | `#F8FAFC` | Text on dark backgrounds |

### Danger Level Colors

These are the canonical colors used in the API and all visualizations:

| Level | Hex | RGB |
|-------|-----|-----|
| VERY_LOW | `#22C55E` | 34, 197, 94 |
| LOW | `#3B82F6` | 59, 130, 246 |
| MODERATE | `#EAB308` | 234, 179, 8 |
| HIGH | `#F97316` | 249, 115, 22 |
| VERY_HIGH | `#EF4444` | 239, 68, 68 |
| EXTREME | `#1A0000` | 26, 0, 0 |

## Typography

- **Headings**: System UI / Inter / -apple-system, 700 weight
- **Body**: System UI, 400 weight
- **Code**: JetBrains Mono / monospace

## Usage Guidelines

- Always maintain clear space around the logo (minimum: half the flame icon height)
- Do not rotate, skew, or distort the logo
- Do not change the logo colors — use the provided variants
- The flame icon can be used alone at small sizes (favicons, social avatars)
- When referencing the project, use "INFERNIS" in all caps for the brand name
